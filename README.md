# 📊 brModelo v2.0 - Versão Estável (Nativa)

Este repositório contém o executável da ferramenta **brModelo** utilizado no meu tutorial de Modelagem de Dados no YouTube.

### 🚀 Por que usar esta versão?
Diferente das versões mais recentes, esta versão **v2.0** é um executável nativo do Windows.
*   ✅ **Não precisa de Java** instalado.
*   ✅ **Não precisa de instalação**: basta baixar e dar um duplo clique para abrir.
*   ✅ **Leve e Estável**: ideal para aprendizado de MER e DER (entidades, atributos e relacionamentos).

### 📥 Como baixar e usar:
1. Clique no arquivo **brModelo.exe** acima.
2. Clique no botão **Download** (no canto direito).
3. Execute o arquivo.
   * *Nota: Se o Windows exibir um aviso de "Proteção do PC", clique em **"Mais informações"** e depois em **"Executar assim mesmo"**.*
